## 1.6 函数
**函数是对实现某一功能的代码的模块化封装,其定义如下**
```c++
返回值类型 函数名(参数类型 参数名1, 参数类型 参数名2,...,参数类型 参数n){
    执行语句
    ...return 返回值;
}
```
